<objective>
CTRL-06 / SC#2: `villa preflight` gates host fitness for the memory stack — (a) free disk at the rootless Podman volume storage root (where the Qdrant vector index lives) must clear a 1 GiB floor, (b) free memory must clear the embedding-footprint reservation — both with refuse-with-remediation, BLOCK on confident failure, typed-Unknown WARN on unevaluable probes (D-06/D-07). Checks are emitted ONLY when memory_enabled=true; the off path is byte-identical (frozen preflight render goldens are the regression net). The same checks gate the install path so an unfit host is refused before the stack comes up.

Purpose: extends "runs healthy after install" to the memory stack — the host is proven fit before Qdrant/embedder are trusted with the unified-memory envelope.
Output: new `checks_memory.go` pure core + cmd-tier gated emission in the preflight verb and install path.
</objective>
