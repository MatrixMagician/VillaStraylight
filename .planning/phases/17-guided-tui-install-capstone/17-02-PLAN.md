| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-17-02 | Tampering | model name interpolation | mitigate | Model choice constrained to catalog ids from `recommend.Pick` options (no free-text into commands); ContainerArgs builds fixed-arg slices behind the seam. |

| T-17-05 | Denial of Service / unsafe mutation | accidental Enter on Install confirm | mitigate | D-07: Install confirm defaults focus to Cancel; mutation only on deliberate Install+Enter; Esc/Ctrl+C aborts non-zero with no mutation. |
