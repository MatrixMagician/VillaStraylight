`detect.Bytes` contract (from `internal/detect/value.go`):
- `detect.KnownBytes(v uint64, src string) Bytes` — Known=true, carries provenance.
- `detect.UnknownBytes(reason, raw string) Bytes` — Known=false, NEVER a bare 0.
- consumers test `.Known` to distinguish "couldn't determine" from a real zero.

`RenderView` is the recommend->orchestrate handoff: it must carry resolved addr/port
PIECES (e.g. `EmbedAddr`/`EmbedPort`), NOT composed URLs and NOT image literals.
`orchestrate` composes `http://villa-embed:8080/v1` and `http://villa-qdrant:6333`
itself later (the `internal/orchestrate/openwebui.go` `buildOpenWebUIView` precedent —
endpoint URLs are built in orchestrate from resolved values, and the image literal lives
ONLY in orchestrate per D-10).
</interface_context>
