| Boundary | Description |
|----------|-------------|
| upstream registry → seam | A rolling tag (rocm-6.4.4) can be re-pushed to a different image; only the @sha256: digest is trusted. |
| hand-edited config.toml → BackendFor | The backend string is untrusted user input; an unknown value must not reach the privileged kfd/dri device path. |
| inference seam → rest of codebase | Image/marker literals must not leak past internal/inference (single-polymorphism guarantee). |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
